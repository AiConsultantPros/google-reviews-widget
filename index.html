How to Set Up

    

      

        
1

        
Deploy the Proxy

        
Deploy the included Cloud Function to Google Cloud Functions. Set your Google API key as an environment variable. Your key stays server-side and is never exposed in client code.

      

      

        
2

        
Find Your Place ID

        
Use the Place ID Finder to look up your business, or call the proxy's /find endpoint with your business name.

      

      

        
3

        
Embed the Widget

        
Copy the embed code from the panel above. Replace the proxy URL and Place ID. Paste it into your website's HTML. Your API key never touches the browser.
